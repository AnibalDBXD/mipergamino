# MiPergamino

# Stack / Tecnologías

NextJS, NextJS API y MongoDB. CSS: Tailwind.

# ¿Deseas colaborar?

https://trello.com/b/MTSwJf4j/miciudad

# Cómo ejecutar proyecto:

```sh
cd mipergamino
```

```sh
npm update
```

```sh
npm install
```

```sh
npm run dev
```

```sh
Renombrar .env.sample a .env y modificar las variables MONGO_USERNAME y MONGO_PASSWORD (pedirmelas).
```
